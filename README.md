# LSPD Card Generator

Generate realistic Los Santos Police Department business cards for GTA World roleplay. Fill in officer details, preview the card live with rank-specific badge images, and export high-quality PNGs or GTAW note-format images.

## Features

- LAPD-inspired card layout with Los Santos branding
- Rank-specific badge images (Officer through Chief of Police)
- Live preview with responsive scaling on mobile
- PNG export (standard and high quality)
- GTAW note export (1000×1415)
- Print support
- Form data persisted in localStorage

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |

## Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- html2canvas (PNG export)
